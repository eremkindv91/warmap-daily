# Публичные схемы данных

Схемы описывают опубликованный snapshot, evidence registry и события. Python-проверки дополнительно контролируют топологию, площадь, задержку публикации, ссылки на источники и пересечения полигонов.

LLM и текстовые адаптеры не создают геометрию. Они могут формировать только документы, claims и кандидаты для ручной проверки.
